## 📦 使用教程

**一、写在最前**

- 不推荐长期使用“**懒人配置**”
- 我只是提供**照葫芦画瓢**的模版
- Surge的**UI编辑**非常完善，所有成熟的功能都可以在UI进行编辑，懂不懂配置文件根本不重要


**二、 开始安装**

1. Surge首页**点击左上角图标**
2. 在**`导入`**中选择**`从URL下载配置`**
4. 返回首页，从**`代理服务器`**中，找到最下面的一/二号机场，输入机场订阅链接，大功告成
