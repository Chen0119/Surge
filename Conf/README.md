## 📦 使用教程

 **一、写在最前**

 - Surge的**UI编辑**非常完善
 - 这只是个**照葫芦画瓢**的模版


 **二、 开始安装**

 1. 前往首页，点击左上角
 2. 选择`从URL下载配置`
 3. 在指定位置，填写订阅链接
