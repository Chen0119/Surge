## 📦 使用教程

**一、写在最前**

- 这只是个**照葫芦画瓢**的模版
- Surge的**UI编辑**非常完善，多试试就会了


**二、 开始安装**

1. 在首页，点击左上角图标
2. 选择`从URL下载配置`
3. 下载完成后，返回首页
4. 点击“代理服务器”，找到"**机场一/二/三**"
5. 在指定位置填写机场的订阅链接
