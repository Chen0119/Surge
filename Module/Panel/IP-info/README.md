## 🚀 功能

- 📍 显示当前节点IP信息
- 📡 显示当前节点运营商信息
- 🌍 显示当前节点所在地信息

## 📘 如何使用

### 🖇 模块链接

> **远程模块：** https://raw.githubusercontent.com/cc63/Surge/main/Module/Panel/IP-info/Moore/IP-info.sgmodule

### 📲 安装方式

1. 打开 **Surge**
2. 选择 **模块** 
3. 点击 **安装新模块...**
4. 复制粘贴上方的安装链接
5. 点击 **完成!**
